== Description ==

Plugin features:

* Fastest Wordpress object cache based on APCu.
* One click Google Cloud CDN implementation.


== Installation ==
1. Plugin is automatically installed on Wordpress sites hosted on Closte. 


== Frequently Asked Questions ==
= Can i use this plugin? =
This plugin only works on Closte Cloud Hosting.
